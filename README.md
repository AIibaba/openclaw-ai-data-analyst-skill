# AI Data Analyst — Turn Raw Data into Business Insights

*Automated data analysis with data cleaning, visualization, statistical analysis, and business insights.*

A powerful AI data analysis assistant that transforms messy datasets into clear, actionable business intelligence. Handles everything from data cleaning and exploratory analysis to statistical modeling and executive-ready visualizations.

## Features

| Feature | Description |
|---------|-------------|
| Data Cleaning Engine | Automatically detects and fixes missing values, outliers, duplicates, and formatting issues |
| Exploratory Analysis | Generates comprehensive statistical summaries, distributions, and correlation analyses |
| Visualization Studio | Creates publication-ready charts, dashboards, and data stories |
| Trend Detection | Identifies patterns, seasonality, and anomalies in time-series and business data |
| Insight Generator | Translates statistical findings into plain-language business recommendations |
| Report Builder | Produces structured analysis reports with methodology, findings, and next steps |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-data-analyst-skill
```

## Usage

```
analyze [dataset]          # Run comprehensive exploratory data analysis
clean [dataset]            # Auto-clean and prepare data for analysis
visualize [metric]         # Create charts and visualizations
trends [time-series]       # Detect patterns and forecast trends
report [analysis]          # Generate a structured analysis report
```
